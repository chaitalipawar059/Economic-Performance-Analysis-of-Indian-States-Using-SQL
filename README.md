# Economic-Performance-Analysis-of-Indian-States-Using-SQL
This project uses SQL to analyze rural unemployment, inflation, and GSDP growth trends across Indian states. The dataset is structured into a relational database to uncover economic patterns and performance insights.

**Business Objectives**
-- Identify states with the highest economic distress based on rural unemployment.
-- Evaluate inflation pressure across states to determine cost-of-living impacts.
-- Analyze state economic performance by assessing GSDP growth consistency over multiple years.
-- Compare inflation trends to highlight top-performing and underperforming states in terms of economic stability.
-- Generate insights for policy-makers and businesses for better resource allocation, investment planning, and economic reforms.
